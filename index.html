Navegación Principal

Bootstrap 5

Definición: Bootstrap 5 es un framework de desarrollo web que facilita la creación de sitios y aplicaciones web responsivas y modernas.

Uso: Se utiliza para diseñar rápidamente páginas web utilizando componentes predefinidos y un sistema de diseño basado en CSS y JavaScript.

Componentes Esenciales

  • Containers: Contenedores fluidos para el contenido del sitio, que aseguran que el contenido esté adecuadamente alineado y con espaciado en el diseño.
  • Grid System: Sistema de cuadrícula flexible que permite crear diseños responsivos utilizando filas y columnas.
  • Flex: Modelo de diseño flexible que ayuda a crear interfaces adaptables con facilidad.
  • Control Carousel: Componente dinámico que permite presentar contenido de manera interactiva mediante un carrusel de imágenes o elementos.
  • Control Navbar: Barra de navegación adaptable que mejora la experiencia del usuario, permitiendo una navegación sencilla y responsiva.

Diseño Web Responsivo

Definición: El diseño web responsivo es una técnica de diseño que permite que un sitio web se adapte automáticamente a diferentes tamaños de pantalla, mejorando la experiencia del usuario sin importar el dispositivo que utilice.

Importancia: Es crucial en la actualidad debido a la gran diversidad de dispositivos utilizados para acceder a internet, desde teléfonos móviles hasta pantallas de escritorio.

Principios Básicos: Uso de layouts fluidos, imágenes flexibles y media queries para ajustar el diseño según el tamaño de la pantalla.

Viewport y CSS3 Media Queries

Viewport

Función: El viewport define el área visible de una página web y cómo se escala en diferentes dispositivos.

Configuración: Se configura en HTML con la etiqueta <meta> y puede controlar el ancho, la escala inicial y otros aspectos de la visualización.

CSS3 Media Queries

Uso: Las media queries permiten aplicar estilos CSS específicos según el ancho, altura, resolución y otros aspectos del dispositivo.

Ejemplos Prácticos:

  • En el CSS, se utilizan media queries para adaptar el diseño a pantallas pequeñas, medianas y grandes.
  • Por ejemplo, se cambia la dirección del layout y el tamaño de las imágenes según el tamaño de la pantalla.
